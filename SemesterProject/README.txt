This is the working directory for the Mobile Currency Reader worked on during the Semester Project.

The directories are as follows:

- ImageAnalysis - Contains examples of the usage of the harris corner detection algorithms with images taken from a camera.  The camera resolution matches the resolution expected from a mobile phone.

- src - the src directory contains the source code for the MCR project including the source that runs on the mobile phone, and the code that runs on the server application.

- Class Diagrams and Enterprise Architect Project - Contains a class diagram of the src dir.
